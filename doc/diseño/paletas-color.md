# Paletas de color

## Cámara Portuaria

| Token | Valor | Uso |
| --- | --- | --- |
| AccentColor | `#1565C0` | Sidebar, botones primarios, encabezados |
| AccentLightColor | `#E3F2FD` | Fondos de selección, áreas destacadas |

## Centro Marítimo

| Token | Valor | Uso |
| --- | --- | --- |
| AccentColor | `#00695C` | Sidebar, botones primarios, encabezados |
| AccentLightColor | `#E0F2F1` | Fondos de selección, áreas destacadas |

## Neutrales compartidos

| Token | Valor | Uso |
| --- | --- | --- |
| BackgroundColor | `#F5F5F5` | Fondo general de la app |
| SurfaceColor | `#FFFFFF` | Tarjetas, paneles, formularios |
| BorderColor | `#E0E0E0` | Bordes de controles y separadores |
| TextPrimaryColor | `#1A1A1A` | Texto principal |
| TextSecondaryColor | `#6B6B6B` | Texto secundario, labels |
| SuccessColor | `#2E7D32` | Confirmaciones, estados positivos |
| WarningColor | `#F57C00` | Advertencias |
| ErrorColor | `#C62828` | Errores, validaciones fallidas |

## Colores de estado de recibo

| Estado | Fondo |
| --- | --- |
| Emitido | `#E3F2FD` |
| Enviado | `#FFF9C4` |
| Pagado | `#E8F5E9` |
| Vencido | `#FFEBEE` |
| Anulado | `#F5F5F5` |

Los recursos están definidos en `Resources/Colors.xaml` de cada proyecto UI.
